# Portfolio project - [Farstad.dev](https://farstad.dev/) 📝

This is a repository for my portfolio! (pretty self-explanatory👀)

This project utilizes front-end build tool _**Vite**_, the frameworks _**TailwindCSS**_ and _**TypeScript**_, as well as the library _**React**_.<br>
The project is hosted on [Farstad.dev](https://farstad.dev/) using Vercel.  🚀

To run the portfolio localy, run the commands: 

    cd portfolio-project
    npm install
    npm run dev

## Technologies used  🛠️
-   [Vite](https://vitejs.dev/)
-   [TailwindCSS](https://tailwindcss.com/)
-   [TypeScript](https://www.typescriptlang.org/)
-   [React](https://reactjs.org/)
-   [Vercel](https://vercel.com/)
